FORMATO DE IMAGENJPEG/JPG

DEFINICIÓN

El JPEG es uno de los tipos de formato de imagen más comunes. Es un formato que suelen utilizar las cámaras digitales, ya que fue diseñado principalmente para uso electrónico. El principal inconveniente de la compresión JPEG es que tiene pérdidas, lo que significa que cada vez que el archivo se edita y se vuelve a guardar en formato JPEG, la calidad se degrada. De hecho, las imágenes con tal degradación de JPEG son comunes en la web hoy en día. Esta degradación, sin embargo, no es el resultado de copiar y compartir el archivo sin recompresión (un error común).

MEJORES USOS

compartir y mostrar imágenes fotográficas electrónicamente y en la web, imprimir fotografías.

VENTAJAS

compatibilidad, tamaños de archivo más pequeños, buena gama de colores.

DESVENTAJAS

la compresión con pérdida degrada la calidad de la imagen cuando se edita varias veces, no es ideal para texto, gráficos simples o ilustraciones.

FORMATO DE IMAGENGIF

DEFINICIÓN

GIF fue uno de los primeros tipos de formato de imagen que se utilizó ampliamente para compartir imágenes y gráficos en la web. Hoy en día, el formato todavía se usa ampliamente para gráficos web simples, ya que tiene una compresión sin pérdidas para líneas simples y colores sólidos para pantallas electrónicas. Aunque poco a poco está siendo reemplazado por el formato de archivo PNG, los GIF siguen siendo relevantes gracias a su principal ventaja: la compatibilidad con animaciones.

Los GIF animados todavía se utilizan ampliamente hoy en día para mostrar y compartir videoclips cortos (obviamente sin sonido) en la web porque el tamaño del archivo es mucho más pequeño que los formatos de video comunes. De hecho, algunos han llamado 2012 el “año del GIF animado”.

MEJORES USOS

Compartir y mostrar imágenes fotográficas electrónicamente y en la web, imprimir fotografías.

VENTAJAS

Compatibilidad, tamaños de archivo más pequeños, buena gama de colores.

DESVENTAJAS

La compresión con pérdida degrada la calidad de la imagen cuando se edita varias veces, no es ideal para texto, gráficos simples o ilustraciones.

FORMATO DE IMAGENPNG

DEFINICIÓN

El PNG era un formato de imagen diseñado específicamente para gráficos web como un reemplazo mejorado no patentado del GIF. Admite la compresión sin pérdida para que no se degrade cuando se edita como con los archivos JPEG. Si bien esto puede crear archivos más grandes para imágenes complejas, como fotografías, las imágenes simples con áreas más grandes de color sólido y líneas más nítidas se pueden comprimir a un tamaño más pequeño con una pérdida limitada de calidad.

Una de las otras ventajas importantes de los archivos PNG es la capacidad de contener transparencia total o parcial (utilizando un canal alfa añadido). Este tipo de formato de imagen brinda a los webmasters y diseñadores más flexibilidad para crear gráficos y diseños dinámicos en la web. Aunque a diferencia de los GIF, los archivos PNG no admiten animación.

MEJORES USOS

Gráficos web simples como logotipos, ilustraciones o renderizado de texto rasterizado.

VENTAJAS

Compresión sin pérdidas, archivos de tamaño pequeño para gráficos simples, renderizado suave para líneas nítidas, soporte de transparencia.

DESVENTAJAS

Los archivos de imagen más complejos y más grandes pueden ser grandes, solo admiten colores web (RGB), compatibilidad limitada.

FORMATO DE IMAGENTIF/TIFF

DEFINICIÓN

TIFF es un formato de archivo de imagen que se utiliza generalmente para trabajar con datos de imagen sin procesar. Inicialmente se desarrolló en la década de 1980 para guardar imágenes de documentos, un poco como la forma en que muchas personas usan archivos PDF ahora. Hoy en día, TIFF sigue siendo un formato de archivo preferido para las personas que necesitan trabajar con imágenes de alta calidad, como artistas visuales y archiveros.

A diferencia de JPEG, TIFF tiene compresión sin pérdidas. Lo único que significa es que cuando el archivo se edita o se guarda, el programa de edición que está utilizando no arroja pequeños fragmentos de información para hacer el archivo más pequeño.

Si bien algunas cámaras digitales usan el formato TIFF, la compresión JPEG se utiliza aún más ampliamente. Sin embargo, muchas de estas cámaras también pueden guardar usando formatos de archivo de imagen sin procesar, que se pueden renderizar como archivos TIFF para preservar la calidad de la imagen.

MEJORES USOS

Trabajar con fotografías sin pérdida de calidad de imagen, impresión.

VENTAJAS

Compresión sin pérdidas, imágenes de alta calidad, compatible con PC y Mac, se pueden guardar varias imágenes y varias páginas en un archivo.

DESVENTAJAS

Archivos de gran tamaño, no excelentes para gráficos web, problemas menores de compatibilidad de software.

FORMATO DE IMAGENBMP

DEFINICIÓN

En la computación, un mapa de bits es literalmente el mapeo de bits, o pequeñas piezas de información en su forma más simple: ceros y unos. Una imagen de mapa de bits generalmente se refiere a la matriz de píxeles de cualquier imagen.

Los archivos BMP, entonces, son esencialmente imágenes sin compresión: contienen datos para cada píxel individual de la imagen. Y dado que el formato de archivo es tan ampliamente compatible, es un formato ideal para trabajar con programas de edición de imágenes. Aunque algunos formatos BMP admiten la compresión (sin pérdida y con pérdida), en los formatos sin formato (sin comprimir) el mayor inconveniente son los tamaños de archivo relativamente masivos.

MEJORES USOS

Edición de imágenes.

VENTAJAS

Sin comprimir para imágenes perfectas, compatibilidad casi universal, compresión eficiente (ZIP, etc.).

DESVENTAJAS

Tamaños de archivo masivos.

FORMATO DE IMAGENPSD

DEFINICIÓN

Como sugiere el nombre, el formato de imagen PSD es un formato de imagen que se utiliza para trabajar con ficheros de imagen y guardar documentos de imagen y trabajos en curso con Adobe Photoshop.

No es uno de los tipos de formato de imagen seguros para la web, por lo que no es compatible con navegadores ni editores o visores de imágenes estándar.

MEJORES USOS

Edición de imágenes.

VENTAJAS

Todos los filtros, transparencias, rutas y ediciones son totalmente personalizables y reversibles.

DESVENTAJAS

Pueden ser incomodamente grandes.

FORMATO DE IMAGENRAW

DEFINICIÓN

Una imagen RAW es uno de los tipos de formato de imagen menos procesados esta lista; a menudo es el primer formato que hereda una imagen cuando se crea. Cuando tomas una foto con tu cámara, se guarda inmediatamente en un formato de archivo sin procesar. Solo cuando cargas su contenido multimedia en un dispositivo nuevo y lo editas con un software de imagen, se guarda con una de las extensiones de imagen explicadas anteriormente.

Las imágenes RAW son valiosas porque capturan cada elemento de una foto sin procesar y perder pequeños detalles visuales. Sin embargo, eventualmente querrá empaquetarlos en un tipo de archivo ráster o vectorial para que puedan ser transferidos y redimensionados para varios propósitos.

Las imágenes sin procesar no están diseñadas para la web o para compartir y no son compatibles con ningún navegador o visor de imágenes importante.

MEJORES USOS

Fotógrafos profesionales, pero hoy muchos smartphones ofrecen la posibilidad de guardar fotografías hechas con las cámaras en este formato.

VENTAJAS

Imágenes de mayor calidad con más variabilidad de colores.

DESVENTAJAS

Archivos de imagen gigantes (un archivo sin formato puede tener fácilmente hasta 20 a 40 megabytes).

FORMATO DE IMAGEN Y TEXTOPDF

DEFINICIÓN

Probablemente asocies PDF con almacenar, guardar y leer documentos basados ​​principalmente en texto. Eso es perfectamente comprensible: el documento está ahí en el nombre después de todo.

Pero los archivos PDF en realidad se basan en el mismo lenguaje PostScript que alimenta los archivos de imágenes vectoriales EPS y también se pueden usar para guardar imágenes e ilustraciones.

Es el tipos de formato de imagen elegido para almacenar ilustraciones, portadas de revistas y más para su posterior impresión.

MEJORES USOS

Para firmar documentos oficiales, para almacenamiento masivo de documentación, para crear facturas y presupuestos y pacer un PDF de cualquier otro tipo de archivo.

VENTAJAS

El texto indexable y de búsqueda lo hace perfecto para infografías o informes detallados. Puede incluir enlaces, botones de CTA y otros elementos interactivos. La escalabilidad no tiene pérdidas.

DESVENTAJAS

No se puede incluir en el contenido web, debe cargarse y leerse como un archivo separado.

FORMATO DE IMAGEPS

DEFINICIÓN

Técnicamente, el formato de imagen EPS no son gráficos vectoriales puros. Como archivo “PostScript”, pueden contener imágenes vectoriales y rasterizadas e incluso ambas en el mismo archivo (muy parecido a SVG). Los PostScripts están diseñados específicamente para manejar gráficos vectoriales para la salida (impresión, etc.) debido a la capacidad de hacer una rasterización sobre la marcha (convertir líneas en píxeles).

Como tal, los archivos EPS suelen ser el tipo de formato de imagen preferido para enviar gráficos vectoriales para imprimir, no solo porque los documentos PostScript están diseñados para la impresión; pero también es ampliamente compatible y flexible. Otros formatos vectoriales generalmente requieren el paquete de software específico en el que fueron creados para abrirlos, donde los archivos EPS son más universales.

MEJORES USOS

Para firmar documentos oficiales, para almacenamiento masivo de documentación, para crear facturas y presupuestos y pacer un PDF de cualquier otro tipo de archivo.

VENTAJAS

Compatibilidad para gráficos vectoriales, ideal para impresión / salida.

DESVENTAJAS

Puede perder el control de las funciones avanzadas de edición de vectores.
