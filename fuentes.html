Formato de fuentes para web

Lo primero que tienes que entender es que las tipografías son archivos descargables e instalables. No todos los formatos de tipografía son compatibles con todos los sistemas operativos o con los lectores web.

También existen fuentes locales y fuentes online. Las fuentes locales son las que se instalan directamente en el ordenador, mientras que las fuentes online son las que puedes instalar en un servidor y linkear a tu página web, aplicación etc.

FORMATOTrueType Font – .TTF

DEFINICIÓN

Es uno de los formatos de tipografía más conocidos. El TrueType fue desarrollado por Apple y ahora se usa para ambos sistemas operativos, tanto Microsoft como Apple. Es uno de los formatos estándar en estos sistemas operativos, la ventaja es que tiene una buena optimización de las fuentes tanto para formatos impresos como para formatos digitales.

FORMATOOpenType Font – .OTF

DEFINICIÓN

Fue desarrollado por Microsoft y Adobe como una extensión para el TrueType de Apple, se podría decir que fue la evolución de TrueType y, con el tiempo, muchas empresas han empezado a elaborar fuentes con este tipo de formato.

Este formato es compatible con varios programas, lo cual le da una ventaja. Pero su característica más importante es la versatilidad que tiene, pues acepta caracteres especiales, lo que permite que tengas varios alfabetos. Al ser aceptada en varias plataformas, algunos de esos caracteres especiales también son aceptados.

FORMATOWindows Font File – .FNT

DEFINICIÓN

Lo mencionamos por su historia, pero este es uno de los formatos de tipografía que ya está obsoloto. Era un formato para almacenar fuentes genéricas del sistema operativo de Windows.

FORMATOEmbedded OpenType – .EOT

DEFINICIÓN

Es un formato de tipografía desarrollado por Microsoft, solo es compatible con Internet Explorer y no se agregó a Microsoft Edge. Este formato se usaba para internet y tenía funciones que prevenían que las personas copiaran las fuentes que tenían copyright. También usaba las fuentes que tenía guardadas para mostrar una página web en caso de que algo le hubiera pasado a los archivos de fuente de las páginas.

FORMATOWeb Open Font Format – .WOFF

DEFINICIÓN

Dentro de los formatos de tipografía para páginas web, está el formato estándar WOFF. Es un formato desarrollado desde las fuentes OpentType y TrueType, sin embargo, se les agrega un formato de compresión específico y se agrega metadata para que sean compatibles con la web.

El motivo de esto era para crear una diferenciación entre las fuentes que pertenecen a los ordenadores y las fuentes de la web, para que las páginas web lograran cargar más rápido y con los metadatos lograban crear licencias a las tipografías.

WOFF 2.0 es el predecesor de WOFF, el cambio más significativo es la capacidad de compresión frente a su antecesor, lo que hace que las páginas carguen mucho más rápido.

FORMATOScalable Vector Graphics – .SVG

DEFINICIÓN

La ventaja del SVG es que es un formato vectorial, lo cual le da más libertad al usuario para moverlo de la manera que prefiera y tiene un peso menor con buena calidad. Es un formato estándar abierto y es compatible en este momento con todos los navegadores. Se usa más para familias tipográficas decorativas, es decir, que tengan adornos y diseños.
